作品说明:

【项目名称】EdgeForm - 边缘表单构建器

【项目简介】
EdgeForm是一个完全运行在阿里云ESA边缘的表单构建和管理系统,充分利用ESA Pages + 边缘函数 + EdgeKV三大核心能力,实现了零后端部署、全球加速和安全可靠的表单解决方案。

【创意卓越】
1. 可视化表单构建器 - 点击添加字段,实时预览,所见即所得的表单设计体验
2. 一键生成嵌入代码 - 自动生成完整的HTML/JS代码,可直接嵌入任何网站,无需额外开发
3. 现代化UI设计 - 深色主题,渐变色彩,流畅动画,提供优秀的用户体验
4. 智能字段管理 - 支持文本、邮箱、电话、网址、数字、日期、下拉选择、文本域等8种字段类型

【应用价值】
1. 开箱即用 - 部署后即可使用,无需复杂配置,降低使用门槛
2. 真实场景应用 - 适用于问卷调查、用户反馈、活动报名、联系表单等多种实际场景
3. 数据管理完善 - 支持数据查看、导出CSV、删除等完整的CRUD操作,满足实际业务需求
4. 零成本运营 - 基于ESA边缘计算,无需维护服务器,运营成本极低
5. 全球可用 - 依托ESA全球节点,任何地区都能快速访问,响应时间毫秒级
6. 开放集成 - 支持嵌入任何网站,提供公开API,支持跨域调用,易于集成

【技术探索】
1. 完整的ESA生态应用 - 深度整合ESA Pages(静态托管) + Edge Functions(4个边缘函数) + EdgeKV(边缘存储),展示ESA生态的完整能力
2. 边缘计算最佳实践 - 展示了如何在边缘构建完整的全栈应用,包括前端、API、数据存储和认证
3. 无服务器架构 - 真正的Serverless,自动扩展,按需付费,零运维成本
4. 现代前端技术栈 - React 18 + TypeScript + Vite,类型安全,开发体验优秀
5. 企业级安全机制 - JWT认证 + SHA-256密码加密 + HMAC-SHA256签名,符合企业级安全标准

【核心功能】
- 表单构建: 支持8种字段类型,字段验证配置(必填、长度限制、数值范围、正则表达式)
- 数据管理: JWT认证保护,数据列表展示,CSV导出,单条删除
- 安全特性: 密码哈希存储,JWT Token认证(7天有效期),权限控制,CORS配置

【技术亮点】
- 4个Edge Functions: submit(表单提交)、auth(认证)、get-submissions(获取数据)、delete-submission(删除数据)
- EdgeKV存储: 表单提交数据、管理员密码哈希、JWT密钥
- Web Crypto API: 密码哈希和JWT签名验证
- 全球边缘节点部署,访问速度快,零冷启动

本项目完整展示了如何利用阿里云ESA生态构建实用的边缘应用,具有创意性、实用性和技术深度。
